## 🚀 Featured Projects

### 🐊 [CrocoManager](https://github.com/tobias-hstbrg/CrocoManager)
A C# school project simulating a wildlife management system for a crocodile and alligator research station in South Florida, designed to showcase OOP architecture and design decisions.

- **Frontend:** C# + .NET MAUI — custom pages with a fully overhauled UI design
- **Services:** Supabase for data and auth (BaaS), NOAA + USGS APIs for weather and water data displayed on a research page with attached environmental observations
- **Testing:** XUnit unit tests against a platform-agnostic core library (since .NET MAUI can't be tested directly)
- **CI:** GitHub Actions — builds an APK and Windows executable from master, and runs the XUnit test suite

### 📦 [shipping-tracker](https://github.com/tobias-hstbrg/shipping-tracker)
A full-stack parcel tracking web app built to learn Spring Boot and React in a real project setting.

- **Backend:** Java 21 + Spring Boot, RESTful API, rate limiting with Bucket4j, Docker
- **Frontend:** React 18 + TypeScript, Tailwind CSS, shadcn/ui, interactive map with MapLibre GL JS
- **DevOps:** Docker Compose, Dev Containers (VSCode), Gradle

### 🌐 [fiber-watcher](https://github.com/tobias-hstbrg/fiber-watcher)
An automated bot that monitors a city website for fiber optic infrastructure updates and sends email alerts.

- **Stack:** TypeScript, Node.js, Cheerio (web scraping), Mailgun
- **CI/CD:** Runs on a schedule via GitHub Actions with JSON-based content caching
